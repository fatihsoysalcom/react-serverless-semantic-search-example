# React Serverless Semantic Search Example

This example demonstrates how to integrate serverless and ultra-fast semantic search capabilities into a React application. It uses a client-side simulation of a semantic search API, showcasing an input field, real-time results, and loading states without a dedicated backend server.

## Language

`html`

## How to Run

Save the code as `index.html`. Open the `index.html` file in your web browser.
